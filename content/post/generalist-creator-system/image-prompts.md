# 文章配图生成 Prompt

生成后将图片保存到本目录，格式为 .webp 或 .png（会自动转为 .webp），文件名与文章中引用一致。

---

## 1. cover.webp — 文章封面

Create a clean tech illustration for an article about multi-interest generalists turning curiosity into a creator system in the AI age. Deep dark navy background #07090f, elegant network of glowing teal #2dd4bf nodes connecting different domains: writing, code, design, business, psychology, learning. One central human silhouette or abstract creator desk, surrounded by modular knowledge blocks flowing into a coherent system. Premium editorial style, minimal and elegant, high contrast, subtle amber #f59e0b highlights, no clutter, dark mode aesthetic, Ghost Protocol palette. No text. 16:9 aspect ratio.

---

## 2. industrial-age.webp — 工业时代流水线与 AI 时代连接网络

Split-scene clean tech illustration. Left side: muted industrial assembly line with identical workers and rigid conveyor belts, gray-blue tones. Right side: dynamic AI-era network with one creator connecting multiple glowing knowledge nodes, teal #2dd4bf and amber #f59e0b accents. Background deep dark navy #07090f, subtle grid, elegant contrast. The mood should show a transition from narrow specialization to connected generalist thinking. No text. 16:9 aspect ratio.

---

## 3. second-renaissance.webp — 第二次文艺复兴中的多领域连接

Create a dark mode renaissance-meets-AI illustration. A modern polymath workspace with hints of anatomy sketches, code windows, design wireframes, books, and neural network diagrams floating together. Use deep navy #07090f background, teal #2dd4bf primary glow, purple #a78bfa and amber #f59e0b secondary accents. Clean tech illustration, minimal and elegant, not historical cosplay, more like a futuristic editorial metaphor for the second renaissance. No text. 16:9 aspect ratio.

---

## 4. creator-vessel.webp — 兴趣到信任的创作者容器

Illustrate a creator system as a vessel: streams labeled visually by icons only (books, notes, code, microphone, product box) flowing into a central transparent container, then flowing outward as content, trust, and products represented by abstract cards and network signals. Deep dark navy #07090f, teal #2dd4bf primary accent, amber #f59e0b highlights, clean tech illustration, elegant and minimal, high-end blog visual. No text. 16:9 aspect ratio.

## 5. idea-museum.webp — 想法博物馆与观点密度

Create an abstract "idea museum" visual: a dark gallery or library with floating luminous idea artifacts, note cards, book fragments, diagrams, and social post cards arranged as curated exhibits. A single creator figure observes and connects them with thin teal lines. Background #07090f, teal #2dd4bf, amber #f59e0b, purple #a78bfa accents. Clean tech editorial illustration, minimal, sophisticated, dark mode aesthetic. No text. 16:9 aspect ratio.

---

## 6. system-product.webp — 系统经济中的个人产品闭环

Create a clean dark mode diagram-like illustration of a personal product system loop: problem -> experiment -> content -> audience -> product -> feedback, represented with icons and flowing arrows but no readable text. Center: elegant modular system dashboard or flywheel. Deep navy #07090f background, teal #2dd4bf primary, amber #f59e0b highlights, red #f87171 sparingly for friction points. Minimal and elegant, premium tech blog style, no text. 16:9 aspect ratio.

---

## 使用说明

1. 将上述 prompt 分别输入 AI 图片生成工具（如 Midjourney、DALL-E、Ideogram 等）
2. 建议使用 16:9 比例，分辨率至少 1920x1080
3. 生成后保存到本目录（PNG 或 WebP 均可，后续会统一转为 WebP）
4. 文章中已经用 `![描述](文件名.webp)` 格式引用了这些图片
