# 文章配图生成 Prompt

生成后将图片保存到本目录，格式为 .webp 或 .png（会自动转为 .webp），文件名与文章中引用一致。

---

## 1. cover.webp — 文章封面

Create a clean tech illustration for a blog post about exporting AI coding conversations from terminal tools into an Obsidian knowledge vault. Show a dark terminal on the left, flowing structured conversation cards into a glowing Obsidian-style vault or graph on the right. Deep dark navy background #07090f, teal accent #2dd4bf, small amber highlights #f59e0b, elegant minimal composition, premium developer tool aesthetic, subtle grid lines, no brand logos. Text is allowed if it improves clarity; keep it short and legible. 16:9 aspect ratio.

## 使用说明

1. 将上述 prompt 分别输入 AI 图片生成工具（如 Midjourney, DALL-E, Ideogram 等）。
2. 建议使用 16:9 比例，分辨率至少 1920x1080。
3. 生成后保存到本目录（PNG 或 WebP 均可，后续会统一转为 WebP）。
4. 文章中已经用 `![描述](文件名.webp)` 格式引用了这些图片。
