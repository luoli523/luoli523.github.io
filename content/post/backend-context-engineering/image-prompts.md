## Prompt (英文)

A cinematic dark-themed illustration representing "backend context engineering" for AI agents. Central visual: a sleek, abstract pipeline or data conduit splitting into two paths — the left path is chaotic, fragmented, with scattered broken data blocks and dim amber glow symbolizing inefficient token consumption and retry loops; the right path is structured, clean, with aligned metadata blocks flowing smoothly in bright teal cyan light representing efficient, well-engineered context. Floating translucent JSON-like structured data fragments, neural network nodes subtly glowing teal (#2dd4bf) as the primary accent, with amber (#f59e0b) as secondary accent highlighting the contrast. Deep space background (#07090f), minimalistic geometric composition, high-tech futuristic aesthetic, no text, no words, no letters, 16:9 aspect ratio, ultra sharp, editorial tech-magazine cover quality, subtle depth of field, glassmorphism and soft light bloom.
